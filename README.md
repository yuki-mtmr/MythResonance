# Myth Resonance

10個の質問に回答すると、ユーザーに適合する神話の神（1柱）を導き出す診断Webアプリです。

## 特徴
- **静的Webアプリ**: Vite + React + TypeScript による高速な動作。
- **GitHub Pages 公開**: GitHub Actions による自動デプロイをサポート。
- **プレミアムデザイン**: ダークテーマ、ゴールドとパープルのアクセント、洗練されたアニメーション。
- **共有機能**: X（Twitter）への結果共有およびクリップボードコピー機能を搭載。

## 開発と実行

### ローカル起動
```bash
npm install
npm run dev
```

### ビルド
```bash
npm run build
```

### デプロイ
`main` ブランチにプッシュすると、GitHub Actions がトリガーされ、自動的に GitHub Pages にデプロイされます。

## 技術スタック
- React 18
- TypeScript
- Vite
- React Router (HashRouter)
- Vanilla CSS
